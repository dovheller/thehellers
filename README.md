# דער אחראי — חבורת ההעלער'ס

זימון אישי, שיחת חיזוק אצל הבית אהרן הקדוש זי"ע, ודף צליבות לכל בני החבורה.

## מבנה

| קובץ | תפקיד |
|------|--------|
| `index.html` | המעטפה החתומה + הזימון האישי (אַרײַן־זאָגן) |
| `shtibel.html` | השטיבל — צ'אט קוויטל עם ה"בית אהרן" + שמועס אד״ש שליט״א ברקע |
| `mikve.html` | אחר השיחה (גברים) — דף הצליבות + מקווה |
| `weiberschul.html` | אחר השיחה (נשים) — דף הצליבות + מטבחון |
| `assets/people.js` | רישום מרכזי של בני החבורה — שם, מין, תפקיד, זימון אישי, צליבות |
| `assets/shmuess_rosh_chodesh_sivan.mp3` | שמועס אד״ש שליט״א — ר״ח סיון תשפ״ו |

## הערה חשובה — הצ'אט בשטיבל

הצ'אט עם "הבית אהרן" משתמש ב־`window.claude.complete` — כלי AI שזמין **רק בסביבה של Claude**. כשהדף רץ ב־GitHub Pages או בכל סטטיק־הוסטינג רגיל, הצ'אט יציג הודעת fallback מנומסת ולא יוכל לענות.

אם רוצים שהצ'אט יעבוד גם בסטטיק־הוסטינג, צריך לחבר API חיצוני (Anthropic, OpenAI וכד׳) דרך פונקציית serverless.

## ניווט בין הדפים

1. `index.html` → בחירת שם → קלף הזימון
2. כפתור CTA → `shtibel.html?who=<id>` 
3. קישור בתחתית השטיבל → `mikve.html?who=<id>` (גברים) או `weiberschul.html?who=<id>` (נשים)
4. כל דף שולח אוטומטית את המין הנכון אם מגיעים אליו בטעות

## פרטיות

תיקיית `uploads/` (לא נכללת ב־git) מכילה את חומר המקור הפרטי: יומן הצ'אט המשפחתי, ה־PDFים, וטיוטות מוקדמות. **אסור** לדחוף אותה לרפו ציבורי.
